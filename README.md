# Autoware V2I Interfaces

ROS 2 packages for integrating Vehicle-to-Infrastructure (V2I) messages with Autoware.

This repository provides bridges, converters, and visualization tools for SPaT, SDSM, and MAP messages, enabling infrastructure sensors and traffic signal controllers to interact with autonomous vehicles running Autoware.
